## ðŸ”‘ Core Concepts

**MCP Server**: All tool invocations flow through the Model Context Protocol server which provides governance, validation, and audit logging.

**Tools**: Implement ITool interface with type-safe schemas

**Agents**: Implement IAgent interface, use MCP to discover and invoke tools

**Adapters**: Bridge external frameworks (LangChain, etc.) to EA-SDK
